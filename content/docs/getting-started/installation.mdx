## Prerequisites

- **Node.js** 18 or later
- **npm** (included with Node.js)
- An AI coding agent (Claude Code, Codex CLI, Gemini CLI, Cursor, etc.)

## Install DexCode

<Steps>

<div>
**Clone the repository**

```bash
git clone git@github.com:co-r-e/dexcode.git DexCode
cd DexCode
```
</div>

<div>
**Install dependencies**

```bash
npm install
```
</div>

<div>
**Start the development server**

```bash
npm run dev
```

Your slides will be available at `http://localhost:3000`.
</div>

</Steps>

## Verify Installation

Open your browser and navigate to `http://localhost:3000`. You should see the default DexCode slide deck with sample slides.

<Callout type="tip" title="Hot Reload">
DexCode uses Vite's HMR (Hot Module Replacement). Any changes to MDX slide files are reflected instantly in the browser — no manual refresh needed.
</Callout>
