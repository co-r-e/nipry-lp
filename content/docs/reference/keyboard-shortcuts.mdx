---
title: Keyboard Shortcuts
description: Quick reference for all keyboard shortcuts in DexCode's slide viewer.
---

## Slide Navigation

| Shortcut | Action |
|---|---|
| `→` or `↓` | Next slide |
| `←` or `↑` | Previous slide |
| `Space` or `Enter` | Next slide |
| `Home` | First slide |
| `End` | Last slide |

## Presentation

| Shortcut | Action |
|---|---|
| `f` | Toggle fullscreen |
| `Esc` | Exit fullscreen |
| `?` | Show keyboard shortcut help |

<Callout type="info">
Keyboard shortcuts are disabled when focus is on input fields, text areas, or interactive elements like buttons and links.
</Callout>
