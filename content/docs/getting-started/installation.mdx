## Prerequisites

- **Node.js** 18 or later
- **npm** (included with Node.js)
- An AI coding agent (Claude Code, Codex CLI, Gemini CLI, Cursor, etc.)

## Install Amaroad

<Steps>

<div>
**Clone the repository**

```bash
git clone git@github.com:co-r-e/amaroad.git amaroad
cd amaroad
```
</div>

<div>
**Install dependencies**

```bash
npm install
```
</div>

<div>
**Start the development server**

```bash
npm run dev
```

The homepage is available at `http://localhost:3000`, listing all decks in the `decks/` directory.
</div>

</Steps>

## Verify Installation

Open your browser and navigate to `http://localhost:3000`. You should see the Amaroad homepage listing available decks. Click on a deck name (e.g., `sample-deck`) to view it.

<Callout type="tip" title="Hot Reload">
Amaroad uses Next.js HMR (Hot Module Replacement). Any changes to MDX slide files are reflected instantly in the browser — no manual refresh needed.
</Callout>

- **`decks/`** — Each subdirectory is a separate deck with its own `deck.config.ts` and MDX slide files.
- **`deck.config.ts`** — Per-deck configuration (title, theme, logo, copyright, etc.).
