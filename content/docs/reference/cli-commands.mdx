## Commands

DexCode is managed through npm scripts defined in `package.json`.

## Development

### `npm run dev`

Start the development server with hot module replacement.

```bash
npm run dev
```

## Build & Export

### `npm run build`

Build the deck for production deployment.

```bash
npm run build
```

### `npm run export:pdf`

Export the deck to a PDF file using Playwright.

```bash
npm run export:pdf
```

### `npm run export:pptx`

Export the deck to PowerPoint format.

```bash
npm run export:pptx
```

## Sharing

### `npm run tunnel`

Start a Cloudflare Tunnel to share the dev server publicly.

```bash
npm run tunnel
```

This requires `cloudflared` to be installed. The command starts the dev server and creates a public tunnel URL.

## Utilities

### `npm run new`

Create a new slide file:

```bash
npm run new slide-04
```

This creates `slides/slide-04.mdx` with default frontmatter.

### `npm run reorder`

Renumber slide files to maintain sequential ordering:

```bash
npm run reorder
```

<Callout type="info">
All DexCode commands are standard npm scripts. You can inspect and modify them in `package.json`.
</Callout>
