---
title: Keyboard Shortcuts
description: Quick reference for all keyboard shortcuts in DexCode's slide viewer and presenter mode.
---

## Slide Navigation

| Shortcut | Action |
|---|---|
| `→` or `Space` | Next slide |
| `←` | Previous slide |
| `Home` | First slide |
| `End` | Last slide |
| `1-9` | Jump to slide number |

## Presentation

| Shortcut | Action |
|---|---|
| `P` | Toggle presenter mode |
| `F` | Toggle fullscreen |
| `Esc` | Exit fullscreen or presenter mode |
| `B` | Black screen (toggle) |

## Development

| Shortcut | Action |
|---|---|
| `Cmd/Ctrl + S` | Save current file (triggers HMR) |
| `Cmd/Ctrl + K` | Open search (in docs) |
| `Cmd/Ctrl + Shift + R` | Force reload |

## Presenter Mode

When in presenter mode, additional shortcuts are available:

| Shortcut | Action |
|---|---|
| `T` | Toggle timer |
| `R` | Reset timer |
| `N` | Toggle speaker notes visibility |
| `+` / `-` | Increase / decrease notes font size |

<Callout type="info">
Keyboard shortcuts work in both the main viewer and the audience window. In presenter mode, the presenter window has priority for input.
</Callout>
