## Built-in Components

DexCode slides are MDX files — Markdown with React components. You can use any built-in component directly in your slides without importing.

## Available Components

### Data Visualization

- **BarChart** — Vertical and horizontal bar charts
- **LineChart** — Line charts with multiple series
- **PieChart** — Pie and donut charts
- **AreaChart** — Stacked and filled area charts
- **KpiCards** — Key performance indicator cards with trend arrows

### Layout

- **Columns** — Multi-column layouts (2, 3, or custom widths)
- **CardGrid** — Grid of styled cards
- **ImageLeft / ImageRight** — Image + text split layouts
- **CenteredHeading** — Full-bleed centered heading slide
- **Dashboard** — Multi-widget dashboard layout

### Content

- **CodeBlock** — Syntax-highlighted code with copy button
- **Table** — Styled tables with header formatting
- **Timeline** — Horizontal or vertical timelines
- **Steps** — Numbered step sequences
- **Checklist** — Interactive checklists
- **Blockquote** — Styled quotations

### Visual

- **Icons** — 1000+ Lucide icons available
- **Shapes** — Decorative SVG shapes
- **BigNumber** — Large stat/number display
- **TeamProfiles** — Team member cards with avatars
- **BeforeAfter** — Side-by-side comparison

### Diagrams

- **Flowchart** — Process flow diagrams
- **Pyramid** — Hierarchical pyramids
- **VennDiagram** — Overlapping circles
- **SWOT** — SWOT analysis matrix
- **CycleDigram** — Circular process diagrams

## Usage Example

```mdx
---
layout: two-column
---

<Columns widths="60,40">

# Q3 Revenue Growth

Our revenue grew **45% quarter-over-quarter**, driven by
enterprise adoption and the launch of our Pro tier.

<BarChart
  data={[
    { label: "Q1", value: 820 },
    { label: "Q2", value: 1200 },
    { label: "Q3", value: 1740 },
  ]}
  unit="K"
/>

</Columns>
```
