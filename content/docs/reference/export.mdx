## Export Formats

DexCode supports multiple export formats for different use cases.

## PDF Export

Generate a high-quality PDF using Playwright's headless browser:

```bash
npm run export:pdf
```

### How It Works

1. Playwright launches a headless Chromium browser
2. Each slide is rendered at the configured resolution (default: 1920x1080)
3. Slides are captured as PDF pages in landscape orientation
4. The output is saved to `exports/deck.pdf`

### Configuration

Customize PDF export in `deck.config.ts`:

```typescript
export default defineConfig({
  export: {
    pdf: {
      width: 1920,
      height: 1080,
      format: "landscape",
      margin: { top: 0, right: 0, bottom: 0, left: 0 },
    },
  },
});
```

<Callout type="warning">
PDF export requires Playwright to be installed. Run `npx playwright install chromium` if you haven't set it up yet.
</Callout>

## PPTX Export

Export to PowerPoint format:

```bash
npm run export:pptx
```

Each slide is rendered as a high-resolution image and inserted into a PPTX file. This preserves the exact visual appearance of your slides.

<Callout type="info">
PPTX export creates image-based slides. Interactive elements (animations, hover effects) are captured as static snapshots.
</Callout>

## Static HTML

Build a self-contained static site:

```bash
npm run build
```

The output in the `out/` directory can be deployed to any static hosting service (GitHub Pages, Vercel, Netlify, S3, etc.).

## Image Export

Export individual slides as PNG images:

```bash
npm run export:png
```

This creates one PNG file per slide in `exports/slides/`:

```
exports/slides/
├── slide-01.png
├── slide-02.png
├── slide-03.png
└── ...
```

### Use Cases

- Embedding individual slides in documents or emails
- Social media sharing
- Thumbnail generation
